# Log-archive-tool\
This script will archive and compress logs from /var/log.
## Getting started<br\>
1. **Prerequisties**
    - **Bash**
    - **sudo** access

2. **Installation**
    
    git clone https://github.com/lin001234/Log-archive-tool.git
    cd Log-archive-tool

3. **Usage**

    ./log_archive.sh /path/to/directory
